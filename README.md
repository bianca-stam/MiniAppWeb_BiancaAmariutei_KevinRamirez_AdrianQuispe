# MiniAppWeb_BiancaAmariutei_KevinRamirez
#Formulario de Inscripción Web
## Descripción del proyecto
Este proyecto consiste en una aplicación web que implementa un formulario de inscripción interactivo utilizando HTML, CSS y JavaScript.  
El objetivo principal es validar los datos introducidos por el usuario, mejorar la experiencia de uso y gestionar la información sin necesidad de un backend.
---
#Funcionalidades principales
- Validación de campos en tiempo real.
- Mensajes de error personalizados.
- Previsualización de los datos antes del envío.
- Guardado y recuperación de datos mediante "localStorage".
- Limpieza y recarga del formulario.
---
#Validaciones implementadas
- Nombre y apellido: mínimo de caracteres.
- DNI: validación mediante expresión regular.
- Teléfono y email: formato correcto.
- Número de entradas: rango permitido.
- Aceptación obligatoria de términos y condiciones.
---
#Tecnologías utilizadas
- HTML5: estructura del formulario.
- CSS3: diseño y estilos visuales.
- JavaScript: lógica, validaciones, eventos y almacenamiento local.
---
#Métodos y conceptos investigados
- Expresiones regulares y método ".test()".
- "localStorage" para guardar datos en el navegador.
- Manipulación del DOM ("createElement", "appendChild").
- Gestión de clases dinámicas con "classList.replace()".
